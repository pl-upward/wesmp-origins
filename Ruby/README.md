# pl_upward's custom origins for 1.20.1

## Dependencies
  * Origins
  * Extra Origins (and it's dependency Pehkui)

### This mod adds the following origins:
  * Seedling
  * Ruby
  * Yang
  * Earthen
  * Balloon
  * Creeper
  * Cannibal
  * The Atom
  * Cognizant
  * Orc Warrior

Descriptions and explainations of powers are listed in the origin selection screen.
